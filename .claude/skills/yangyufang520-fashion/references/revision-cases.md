# 作者返稿案例库(活文件,持续追加,永不删除旧条目)

> **这个文件存在的原因**:规则管"不犯错",范文展示"什么是好",但都够不到最细的那层,"差半口气的句子"和"好句子"之间的差距。作者每次手改的"原句→改后"对照,是唯一能校准这一层的材料。规则会互相打架,案例不会。
>
> **用法(硬性)**:
> 1. **动笔前**通读全部案例,带着作者最新的手感校准动笔;
> 2. **交稿前**逐条对照,案例里踩过的坑本篇一个不许再犯;
> 3. 作者每返一处修改(哪怕只改几个字),**当次就追加一条**,格式照下方模板,提炼出通用手感;
> 4. 同一个坑如果第二次被作者改到,说明提炼得不够狠,重写该条的"手感提炼"并加粗标记。

## 记录模板

每条四行:原句(交稿版) / 作者改后 / 差在哪 / 手感提炼(通用化,让下一篇直接可用)。

## 001 · 配饰句的教程腔与主语断裂

来源:2026-07《"裙子+奶奶鞋"今夏太流行了》图12段,作者手改。

- **原句**:腰间再别一方佩斯利丝巾，讲究全藏在细节里。
- **作者改后**:腰间一方佩斯利丝巾，把讲究全藏在细节里。
- **差在哪**:两处。其一,"再别"是教程腔动词,在指挥读者做动作,S姐不使唤读者;其二,后半句主语从丝巾突然跳到"讲究",两个小句之间裂了一道缝,读着不连贯。
- **手感提炼**:
  1. **配饰入句优先"名词短语造景"**,"腰间一方丝巾""脚下一双玛丽珍""头上一顶草帽",让画面自己立起来,少用"再+动词(别/戴/加/系)"向读者下指令;确需动词时,让"穿着的人"或"物件"做主语,不要祈使。
  2. **相邻小句尽量让同一个"物"把动作做完**,把字句是接续主语的好工具("腰间一方丝巾,把讲究全藏在细节里",丝巾是藏的人)。这正是范文"物件做主语+化学动词"句法(蓝衬衫接过接力棒/小碎花降解红色的野)在单句内部的最细粒度形态,检查每个逗号两侧,主语是不是同一个,断了就用把字句或"它"焊回去。

## 002 · 初稿必肥,交稿前必须实测字数

来源:2026-07《"裙子+奶奶鞋"今夏太流行了》创作过程,自查发现(非作者返稿,但教训真实)。

- **现象**:初稿纯汉字 1886 字,而同类范文实测只有 1134~1430 字(文章2/8/11),初稿比范文密度肥了三成。
- **肥在哪**:"这时候""而且""很有…的一套"这类填充性过渡词,一个信息点交代两遍,以及不必要的限定语("家族里的两大主力"→"两大主力")。
- **手感提炼**:交稿前**用工具实测纯汉字数**(不要目测估算),落在 1200~1600 之间;超了先砍填充词与重复交代,不砍信息点;主动把两三段压成四五十字的短段,短段不是偷懒,是范文本来的呼吸。
